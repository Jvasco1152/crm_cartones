# CRM Comercial con Inteligencia Artificial
## Solución diseñada para Cartones América Colombia

---

## El problema que resolvemos

Cada día, tu fuerza comercial invierte tiempo valioso en tareas que deberían ser simples: recordar qué cliente necesita seguimiento, redactar el mismo email una y otra vez, buscar en qué quedó la negociación con determinado cliente, o intentar tener claridad sobre cuánto vale el pipeline de ventas.

**El resultado:** oportunidades perdidas, clientes desatendidos y una comercial agotada haciendo trabajo administrativo en vez de vender.

---

## La solución: CRM Comercial con IA para Cartones América

Un sistema diseñado específicamente para la realidad comercial de una empresa de embalaje en Colombia. No es un software genérico: cada detalle fue pensado para el día a día de quien vende cartón corrugado, cajas de flor, empaques industriales y cartón plegadizo a empresas colombianas.

---

## Qué incluye el sistema

### Panel de Control en Tiempo Real

Al abrir la aplicación, la comercial ve de inmediato lo que importa:

- **Cuántos clientes activos** tiene en su cartera
- **Cuántas oportunidades abiertas** están en el pipeline
- **El valor total del pipeline** expresado en pesos colombianos
- **Cuántas actividades** realizó esta semana
- **Gráfica del pipeline** por etapa de negociación
- **Tareas urgentes** del día con fechas límite
- **Actividades recientes** con un clic para ver detalle

Todo actualizado en tiempo real, sin necesidad de abrir hojas de cálculo.

---

### Gestión de Clientes (Cartera Completa)

Un directorio comercial profesional con toda la información que necesitas:

- Razón social, NIT, ciudad, segmento de industria
- Contacto principal, cargo, teléfono, celular y email
- Estado del cliente: Activo, Inactivo, Prospecto o En Negociación
- Volumen mensual estimado en COP
- Productos de interés (tipo de cartón o empaque)
- Notas internas de la relación comercial
- Historial completo de oportunidades y actividades por cliente

**Búsqueda inteligente** por nombre de empresa, contacto o NIT.

**Filtros** por ciudad (Bogotá, Medellín, Cali, Barranquilla y más), segmento (Alimentos, Flores, Farmacéutico, Retail, Industrial, etc.) y estado.

---

### Pipeline de Ventas — Vista Kanban

Visualiza todas las oportunidades comerciales organizadas en columnas por etapa:

```
Prospección → Contacto Inicial → Presentación → Cotización → Negociación → Ganado / Perdido
```

Cada oportunidad muestra:
- Nombre del negocio y empresa cliente
- Valor en pesos colombianos
- Probabilidad de cierre (%)
- Fecha estimada de cierre
- Tipo de producto (caja corrugada, cartón plegadizo, display, etc.)

El sistema calcula automáticamente el **valor total del pipeline** y la **tasa de cierre histórica**.

También disponible en **vista de lista** para quienes prefieren trabajar en tabla.

---

### Registro de Actividades Comerciales

Un historial cronológico de cada interacción con cada cliente:

- Llamadas telefónicas
- Visitas presenciales
- Correos electrónicos
- Mensajes de WhatsApp
- Reuniones
- Cotizaciones enviadas
- Propuestas presentadas

Cada actividad registra: fecha, descripción de lo ocurrido, resultado (Positivo / Neutral / Negativo / Sin Respuesta) y **la próxima acción acordada con su fecha**.

Esto elimina el "¿en qué quedamos?" de cada conversación.

---

### Tareas y Recordatorios

Una lista de pendientes organizada por prioridad:

- **Alta** — Lo que no puede esperar (cotizaciones por enviar, contratos por firmar)
- **Media** — Seguimientos programados
- **Baja** — Acciones a mediano plazo

Cada tarea puede estar asociada a un cliente específico. Al completarla, un clic la marca como hecha. Las tareas vencidas se resaltan en rojo para que nada se pierda.

---

### Asistente de Inteligencia Artificial

El diferencial más poderoso del sistema: **un asistente de ventas disponible 24/7**, entrenado específicamente en la industria del embalaje colombiano y en técnicas de venta B2B.

Funciona como un asesor comercial experto que conoce:
- El portafolio de productos de Cartones América
- Los sectores cliente del mercado colombiano
- Las temporadas comerciales críticas (San Valentín, Día de la Madre, Navidad)
- Los precios y márgenes del sector en COP
- Las ciudades y dinámicas del mercado nacional

**Lo que puede hacer por ti con un solo clic:**

| Acción | Descripción |
|--------|-------------|
| Analizar tu pipeline | "¿En qué oportunidades debo enfocarme esta semana para cerrar más?" |
| Redactar email de seguimiento | Un correo profesional para el cliente que no respondió la cotización |
| Script de llamada | Guión para llamar a un prospecto del sector farmacéutico |
| Mensaje WhatsApp | Texto corto y efectivo para reactivar un cliente inactivo |
| Estrategia por sector | Tácticas específicas para flores, alimentos, retail o industrial |
| Manejo de objeciones | Argumentos para cuando dicen "está muy caro" |
| Propuesta de valor | Diferenciadores frente a la competencia |
| Preparación para visitas | Qué investigar y qué proponer antes de ir donde un cliente nuevo |

**Contexto inteligente:** Puedes seleccionar un cliente específico y el asistente usará sus datos (empresa, segmento, ciudad, historial) para darte recomendaciones mucho más precisas.

## Pensado para Colombia

El sistema tiene integrado el contexto del mercado colombiano:

- Moneda en **pesos colombianos (COP)** con formato local ($1.500.000)
- Fechas en formato **DD/MM/AAAA**
- **Ciudades colombianas** preconfiguradas: Bogotá, Medellín, Cali, Barranquilla, Bucaramanga, Pereira, Manizales, Cartagena y más
- **Segmentos de industria** relevantes para el embalaje: Alimentos, Bebidas, Flores, Farmacéutico, Industrial, Retail, Textil, Construcción, Logística
- **Productos del portafolio** de Cartones América preconfigurados en el sistema

## Tecnología utilizada

| Componente | Tecnología |
|-----------|-----------|
| Aplicación web | Next.js 16 (moderno, rápido, seguro) |
| Base de datos | SQLite local (sin servidor externo, datos en tu equipo) |
| Inteligencia Artificial | Groq API con modelo Llama 3.3 70B de Meta |
| Diseño | TailwindCSS (interfaz limpia y profesional) |

**No requiere internet para funcionar** (excepto el módulo de IA que consume la API de Groq).

---

## Cómo se instala

```
1. Instalar Node.js en el computador (gratuito)
2. Ejecutar: npm run setup
3. Abrir el navegador en: http://localhost:3000
```

Listo. En menos de 5 minutos el sistema está operativo con datos de ejemplo listos para explorar.
